What it does

  • Pull data across spreadsheets — connect any source spreadsheet by URL and bring its data into a sheet you choose, into an existing tab or a new one.
  • Filter before it lands — combine multiple conditions (equals, contains, greater/less than, is empty, and more) that must all match, so only the rows you need come through.
  • Sort automatically — pick any column to sort by, ascending or descending, applied as part of the same operation.
  • Automate it — set CopydoWell to run on its own: whenever the source sheet is edited, whenever a linked Google Form is submitted, or on a recurring schedule.
  • Mail merge — send a personalized email to every matching row in a sheet, with dynamic attachments and inline images pulled from Google Drive.

Who it's for

Teams who regularly consolidate, clean, or route data between Google Sheets, or send bulk personalized emails from spreadsheet data — without writing formulas or scripts themselves. CopydoWell adds simple sidebars inside Google Sheets that anyone in an organization can use.

Install

CopydoWell is installed from the Google Workspace Marketplace directly into Google Sheets. Once installed, open its tools from the Extensions menu in any Google Sheet:

  • Extensions → CopydoWell → Open Data Tools — copy, sort, filter, and automation
  • Extensions → CopydoWell → Mail Merge — bulk personalized email

Using Data Tools (copy, sort, filter)

Open Extensions → CopydoWell → Open Data Tools. The sidebar walks through four steps, plus optional automation.

1. Destination

Pick a sheet in the spreadsheet you currently have open — this is where the data will land. You can use an existing sheet or create a new one. Choose Overwrite to replace its contents each time, or Append to add rows below what's already there. "Include header row" controls whether the column headers are copied along with the data.

2. Source

Paste the URL (or ID) of the spreadsheet you want to pull data from, then click Load source. Once connected, pick which sheet within that spreadsheet to read.

3. Sort

Optionally choose a column from the source sheet to sort by, ascending or descending.

4. Filter

Add one or more conditions (column, operator, value) — for example "Status equals Active." All conditions must match for a row to be included (AND logic). Available operators: equals, not equals, contains, does not contain, greater than, less than, greater or equal, less or equal, is empty, is not empty.

Click Run now to perform the copy immediately.

Automation

Below Run now, three toggles let a saved job run on its own:

  • Run automatically when the source sheet is edited — re-runs the job whenever the source sheet changes.
  • Run automatically when the linked form is submitted — re-runs the job whenever the source spreadsheet's linked Google Form receives a new response.
  • Run on a schedule — every N minutes, every N hours, or daily at a chosen hour.

Turning on any toggle saves your current Destination/Source/Sort/Filter settings as "the job" and creates the matching trigger. Turn a toggle off at any time to remove that automation.

Using Mail Merge

Open Extensions → CopydoWell → Mail Merge. This tool reads a sheet in your current spreadsheet and sends one email per matching row.

Sheet & email column

Pick the sheet with your recipient list, then pick which column contains each recipient's email address.

Subject & message

Write your subject and message. Use {{ColumnName}} anywhere to insert that row's value — for example: Hi {{First Name}}, your order {{Order ID}} has shipped.

Check Send as HTML to write formatted messages (bold, links, images). With HTML enabled, you can also insert a picture pulled directly from a Google Drive link stored in a column, using {{image:ColumnName}} — for example {{image:Photo Link}}, where the "Photo Link" column holds a Drive share link for each row's image.

Attachments

Choose one of:

  • None — no attachment.
  • Same file for everyone — paste a single Drive file link/ID; it's attached to every email.
  • From a column — pick a column containing a Drive file link per row, so each recipient gets their own file (e.g. a personalized invoice or certificate). Separate multiple links in one cell with commas to attach more than one file per row.

Filter, preview, and send

Optionally add filter conditions so only some rows are emailed (same rules as Data Tools). Use Preview count to see how many recipients currently match and how much of today's sending quota remains, and Send test to myself to send one real email — using the first matching row's data — to your own address before sending to everyone. Send now asks for confirmation showing the exact number of recipients before it sends anything.

Mail Merge sends via your own Google account's daily email quota (Google enforces this limit, not CopydoWell), so very large lists may need to be sent across more than one day.
